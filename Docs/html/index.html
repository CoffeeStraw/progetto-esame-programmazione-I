GiesonFault
Progetto Programmazione I

Introduzione

Il progetto GiesonFault è stato realizzato come progetto per l'esame di Programmazione I al dipartimento di Matematica e Informatica di Perugia.
Il progetto è stato creato con la forte attenzione alla produzione di un codice solido e tecnico. Una forte leggibilità del codice e un'esperienza utente ottimale sono state le priorità in fase di sviluppo.

Compilazione ed avvio

Compilazione:

gcc -o Output main.c gamelib.c -Wall -std=c11

Per entrare in modalità debug, (dove in partenza si avranno 99 oggetti di ogni tipo) è possibile compilare nel seguente modo:

gcc -D DEBUG -o Output main.c gamelib.c -Wall -std=c11

Avvio:

./Output
